# L'excellence selon Marc-Antoine

**Hôtel de Luxe Marrakech : Marc-Antoine, comment définiriez-vous l'hospitalité marocaine aujourd'hui ?**

*Marc-Antoine :* L'hospitalité marocaine est unique car elle est généreuse par nature. Mon rôle est de canaliser cette générosité pour atteindre une précision toute française dans le service. C'est ce mariage qui crée l'exception.

**P&R : Quelle est la place de l'innovation dans un palace historique ?**

*M-A :* L'innovation doit être invisible. Elle doit servir le confort du client sans jamais briser le charme de la tradition. Nous utilisons la technologie pour mieux anticiper les besoins, pas pour remplacer le contact humain.
